# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-12

### Added
- Interactive GitHub username input
- Optional GitHub token support
- Repository table view
- Full tree structure output
- Repository selection by number
- Emoji-based file and folder icons
- Metadata support
- Depth limit support
- Export support for text, markdown, and JSON
